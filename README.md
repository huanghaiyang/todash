# A lodash implement by Dart

## Add Dependency
```yaml
dependencies:
    todash:
        git: git://github.com/huanghaiyang/todash
```

## supports

### Array
+ concat

### Collection
+ slice
+ flatten
+ flattenDeep
+ flattenDepth
+ chunk
+ flatMap
+ flatMapDeep
+ flatMapDepth
+ map
+ each
+ forEach

### Function
+ after
+ before
+ defer
+ negate
+ once
+ overArgs
+ memoize

### Lang
+ isNull
+ isEven
+ isOdd

### Math

### Object
+ get
+ pick       // The type of key should be String, fix for other type later
+ pickBy
