# A lodash implement by Dart

## Add Dependency
```yaml
dependencies:
    todash:
        git: git://github.com/huanghaiyang/todash
```

## supports

+ get
+ slice
+ flatten
+ flattenDeep
+ flattenDepth
+ chunk
+ isNull
+ isEven
+ isOdd
+ after
+ before
+ flatMap
+ flatMapDeep
+ flatMapDepth
+ defer
+ negate
+ square
+ double
+ once
+ overArgs
